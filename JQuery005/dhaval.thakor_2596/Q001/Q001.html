Create one Product Form with Following Field

Name (TextBox)

Description (Text Area)

Price textbox(Type →number)

Image (Type→ File)

Status Active/Inactive (Radio Button)

Category drop down list (Food/Entertainment)

Submit button

 

Below Validation required

None of the field should be empty. 

if all the validation is passed data should be display in tabular format.

use Array Functionality if required.

One Edit Button in table to edit any required

Use jQuery to fetch the values from the control and adding data to the table.
